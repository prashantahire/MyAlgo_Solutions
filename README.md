# MyAlgo_Solutions
#PEAK PROBLEM

This project is for giving solutions to solve one dimensional and two dimensional peak finder problem.
1D Peak definition: a, b, c, d, e b is peak if b>=a and b>=c.
2D Peak definition:

a, b, c, d
e, f, g, h 
i, j, k, l

f is peak if f>=a and f>=g and f>=b and f>=j
